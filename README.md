## Getting Started

- Clone the repository by running `git clone https://github.com/VitorGuedesMadeira/React-Template.git` in your CLI.
- Type `cd React-Template`
- Type `code .`
- Run `npm install`
- Run `npm start`
- Open Browser `http://localhost:3000/`
